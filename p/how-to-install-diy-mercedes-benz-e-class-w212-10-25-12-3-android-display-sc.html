FAQ: (most problems are caused by wiring and settings, please feel free to contact us if you have any problems during installation)

*Android no sound:
1. Check optic cables well relocated correctly and closely.
2. If there is Aux input port on car, connect Audio cable/box into it.
3. Choose AUX Switching mode “Manual” in factory settings, then go to OEM radio and choose “Audio-AUX” , touch screen to android, sound out. (BENZ NTG5 and BMW EVO is in USB function shows USB device connected) .
4. If you want AUX automatic switch, needs to set correct AUX position in android setting for BENZ NTG4.5/4.7/5 , or else no sound, details refer to setting instruction.
5.please check the android factory setting(code is 2018) to check if the "CAN Protocol" chosen is NTG5.0/NTG4.5/NTG4.0/NBT/CIC/CCC, Choose according to original system of the CD（Usually set at the factory, no need to change）
note: if your car is Benz NTG4.5 which has no AUX, need to activate Aux inside factory settings first, the route is : Factory Settings-Vehicle-AUX Activate after restart, you will see AUX inside NTG menu. https://youtu.be/k6sPVUkM9F0 --- Video to show how to activate Aux
for benz NTG4.5/NTG4.0 need to confirm that the white plug connection on the power main cable is correct

https://youtu.be/XEd1lTV1Cjc --- Video for Benz to show how to set AUX Switching mode to “Manual” for sound.
https://youtu.be/QDZnkZIsqIg --- Video for BMW to show how to set AUX Switching mode to “Manual” for sound.
________________________________________________________________________________________
*OEM factory radio menu is not showing correctly or flashing:
In android setting-factory settings-car display, password: 2018, please choose Cartype one by one according to the original radio system such as CCC, CIC, NBT or NTG4.0, NTG4.5, NTG5 , not car models until OEM radio display correct.
https://youtu.be/a6yyMHCwowo --- Video to show how to select Cartype for BMW
https://youtu.be/S18XlkH97IE --- Video to show how to select Cartype for Benz
________________________________________________________________________________________
*No signal: if it is no signal on original radio system, please check the followings:
1. If the LVDS cable is plugged into android screen correctly.
2. If the optic cables are relocated correctly.
3. If the original CD/headunit is turned on.
4. If screen is in AUX mode, it will show “No signal”, choose the right upside corner on screen, it will go back to android
5. please check the android factory setting(code is 2018) to check if the "CAN Protocol" chosen is NTG5.0/NTG4.5/NTG4.0/NBT/CIC/CCC, Choose according to original system of the CD（Usually set at the factory, no need to change）
note: for benz NTG4.5/NTG4.0 need to confirm that the white plug connection on the power main cable is correct
________________________________________________________________________________________
*Camera: if rear camera not works
1. If it is OE camera, just need to choose "OEM camera" in camera type in android setting(System--Camera Selection--OEM Camera).
If it is aftermarket camera, need to choose "aftermarket camera" in camera type in android setting , BMW manual gear car need to go into factory setting to change it from Automatic to Manual.
For wiring aftermarket camera, check camera connection in the paper in the package.(bmw manual and automatic gearbox wiring is different)
2. If still not work: please try all option in Factory setting--Vehicle--gear Selection-gear 1, 2, 3 to check which one makes the camera work
________________________________________________________________________________________
*Carplay connection：If Carplay connection is unsuccessful
If use Carplay, please delete phone Bluetooth record first, turn on phone WIFI, pair Bluetooth to android Bluetooth only, it will go to carplay menu(phonelink in menu or zlink in app)when use carplay, Bluetooth menu shows closed, also android WIFI is turn off. it is correct, refer to https://youtu.be/SqNyvvn4Jjw
still not work, try to re-install z-link, refer to https://youtu.be/VNEE3Yd6VKo
