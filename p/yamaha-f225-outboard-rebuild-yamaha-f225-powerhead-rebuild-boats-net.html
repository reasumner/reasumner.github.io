Here, John Talley takes you through a powerhead rebuild on a 2004 Yamaha F225 from start to finish. We’ll start with the bottom end, and John will show you how to install the crankshaft and pistons. This is a complicated process, since you’ll need to take numbers from the crankshaft, crankcase, and connecting rods to get the right bearing sizes. We’ll also install pumps, hoses, and close the crankcase cover.


HEAD INSTALLATION - 37:25

With the bottom end complete, we’ll show you how to install a top end assembly on a Yamaha outboard. The heads on this unit were shot, and Yamaha offers complete assemblies for the right and left side. We’ll need to remove the cams cam chain tensioner to get this done, and then we’ll get the heads bolted onto the engine. Getting the correct torque is essential here, and John will lay out the precise specs for each bolt.


ELECTRICAL SYSTEM - 49:00

With the heads installed, we’ll turn to the electrical system. John will show you how to install a flywheel on a Yamaha F225, and then we’ll work on installing the camshaft gears and setting the timing. We’ll also install the temperature sensor, regulator rectifier, the wiring harness, and coil packs.


FUEL SYSTEM - 1:05:25

Finally, we’ll install the fuel system on the F225. John will show you how to test the fuel pump on a Yamaha outboard, and then we’ll work on reassembling our VST tank. Then, John will show you how to clean the fuel injectors on an outboard using a kit from Motion Pro.


Then, we’ll install the intakes and make several of the electrical connections as well as get the throttle linkage attached.
