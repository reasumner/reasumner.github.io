5 ปลาไทยโอเมก้า 3 สูง ไม่แพ้ปลาแซลมอน ราคาถูกและดี

หัวข้อที่ 1. มายาคติของ "ปลาเมืองนอก" และความลับของโอเมก้า 3
โอเมก้า 3 โดยเฉพาะ EPA และ DHA คือกรดไขมันจำเป็นที่ร่างกายไม่สามารถสร้างเองได้ แต่มีบทบาทสำคัญในการลดการอักเสบ บำรุงสมองทั้งในเด็กและผู้สูงอายุ หลายคนเข้าใจผิดว่ามีเพียงปลาแซลมอนหรือปลาทะเลน้ำลึกจากต่างประเทศเท่านั้นที่มีโอเมก้า 3 สูง ทั้งที่จริงแล้ว ปลาไม่ได้ผลิตโอเมก้า 3 เอง แต่ได้รับจากอาหารธรรมชาติอย่างแพลงก์ตอนหรือสาหร่าย ซึ่งปลาไทยก็สามารถเข้าถึงได้ไม่ต่างกัน จากข้อมูลของหน่วยงานโภชนาการไทย ปลาไทยบางชนิดมีปริมาณโอเมก้า 3 ที่สูงกว่าหรือเทียบเท่าปลาแซลมอน แต่ราคาถูกกว่า 5–10 เท่า นี่จึงไม่ใช่แค่เรื่องของอาหาร แต่คือการเลือกอย่างฉลาดทั้งด้านสุขภาพและเศรษฐกิจ

หัวข้อที่ 2. เปิดตัว 5 ปลาไทย "ถูกและดี" ที่คุณต้องรู้
ปลาสวายคือแชมป์ที่หลายคนมองข้าม แต่มีโอเมก้า 3 สูงถึงกว่า 2,000 มิลลิกรัมต่อ 100 กรัม บางการศึกษาบอกว่าสูงกว่าแซลมอนด้วยซ้ำ แม้จะมีไขมันรวมสูง แต่หากกินในปริมาณพอเหมาะ ก็เป็นแหล่งไขมันดีชั้นยอด ตามมาด้วยปลาทู ขวัญใจคนไทย ที่มีโอเมก้า 3 ราว 1,600 มิลลิกรัม เป็นปลาผิวน้ำอายุสั้น จึงสะสมโลหะหนักน้อย กินได้ปลอดภัยกว่า ปลาช่อนก็ไม่ควรถูกมองข้าม เพราะนอกจากเนื้อที่อร่อย ยังมีกรดไขมันช่วยสมานแผลและลดการอักเสบ ปลาดุกแม้ถูกมองว่า "สกปรก" แต่หากเลือกจากแหล่งเลี้ยงที่สะอาด ก็เป็นแหล่งโปรตีนและโอเมก้า 3 ราคาประหยัด สุดท้าย ปลาอินทรีหรือปลาจะละเม็ดขาว เป็นปลาทะเลไทยเนื้อแน่น ก้างน้อย ไขมันดีไม่แพ้ปลาเมืองนอก แต่ยังเข้าถึงได้ในราคาที่จับต้องได้

หัวข้อที่ 3. ดาบสองคม: วิธีปรุงเปลี่ยนชีวิต
แม้ปลาไทยจะอุดมด้วยโอเมก้า 3 แต่ “วิธีปรุง” ที่นิยมในไทยอย่างการทอดน้ำมันท่วม กลับทำลายคุณค่านี้จนแทบไม่เหลือ ความร้อนสูงจะทำลายโครงสร้างของกรดไขมันดี และแทนที่ด้วยไขมันเลวร้ายจากน้ำมันทอด กลายเป็นว่ากินแล้วไม่ได้ประโยชน์ แต่ได้โทษแทน ทางออกคือเปลี่ยนมาใช้วิธีนึ่ง ต้มยำ ต้มส้ม หรือย่างไฟอ่อน ซึ่งรักษาโอเมก้า 3 ไว้ได้ดีกว่ามาก อีกปัจจัยสำคัญคือ “แหล่งที่มา” โดยเฉพาะปลาน้ำจืดอย่างปลาดุกหรือปลาสวาย ควรเลือกจากฟาร์มที่ได้มาตรฐานหรือผ่านการรับรองจากกรมประมง เพื่อลดความเสี่ยงจากสารตกค้างหรือโลหะหนัก อาหารดีต้องเริ่มตั้งแต่แหล่งวัตถุดิบจรดวิธีปรุง

สรุป
จากวันนี้ไป อย่ามองปลาไทยในตลาดว่าเป็นเพียงอาหารราคาถูกอีกต่อไป เพราะเบื้องหลังเนื้อปลาตัวละไม่กี่สิบบาทนั้น คือขุมทรัพย์โอเมก้า 3 ที่เทียบชั้นปลาแซลมอนได้อย่างไม่เสียเปรียบ สิ่งที่ผมอยากฝากไว้มีสามข้อหลัก: ข้อแรก “ของดีไม่จำเป็นต้องแพง” การเลือกกินปลาสวายหรือปลาทูไม่เพียงประหยัดเงิน แต่ยังสนับสนุนเกษตรกรไทย ข้อสอง “ความหลากหลายคือกุญแจ” กินสลับระหว่างปลาทะเลและน้ำจืด ปลาเกล็ดกับปลาหนัง จะได้สารอาหารครบและลดความเสี่ยงจากการสะสมสารพิษ และข้อสาม “อย่าฆ่าปลาด้วยน้ำมัน” วิธีนึ่ง ต้ม หรือย่าง คือทางรอดของโอเมก้า 3 ลองเริ่มตั้งแต่มื้อเย็นนี้ ไปตลาด ซื้อปลาทูสักตัว หรือสั่งต้มยำปลาช่อนสักถ้วย แล้วคุณจะรู้ว่า ยาอายุวัฒนะที่แท้จริง อยู่ใกล้ตัวแค่ปลายนิ้วคีบในจานของคุณนี่เอง
