Mercedes W221 from 2006 to 2013. The original was a 7" analogue display with a resolution of only 800*480 pixels. Most of them are now sold as Mercedes w221 android 10.25 screen, Now you can upgrade it to a android 11 (6+128) 12.3" display and juxtapose it with your gauges as a dual screen style.
The installation of w221 android screen 12.3 is a bit longer and a bit more challenging, so you can give it a go if you're not afraid of the hassle.After install, You will get CarPlay and Android auto on the new Mercedes w221 andorid screen.

Cautions.
1. 2006-2009 NTG3.0, if the original car does not have AUX, you can activate it using the activation menu inside the YeeHung product. Otherwise you will not get the Android sound. For the display section you need to select item 22, OEM Display, within the factory settings.
2. 2010-2013 NTG3.5 has AUX menu, you can use it directly to get Android sound, for the display part you need to select item 20 in the factory settings for OEM Display.
3. After installation, the button pad on the left side of the original screen was removed and its function was moved inside the display, you need to open this touch key within the settings at
Settings-Factory Settings-Onboard Configuration-Control Panel (select 3.0 or 3.5 depending on the original system)
