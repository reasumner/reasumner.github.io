Chevy LS3 + Dodge 8HP70 Swapped BMW E90 E91 328xi Wagon

Published at : 12 Dec 2025
We swapped in a fully built LS3 engine with a Dodge RAM 8hp70 transmission. 500whp / 8000 RPM Redline grocery getter.
We built this car to learn and develop parts for it, and of course to have some fun. Follow along as we build this car and continue to update as we make and add changes to the car.
